# _Cypress Simulator_

👋 Welcome!

It's great to have you here. I'm sure you will love this course. ❤️

## What will you learn?

- How to plan which tests should be implemented and in what order
- How to ensure stable tests
- How to handle random waits
- How to handle _captcha_
- How to handle _cookie_ consent banners 🍪
- How to create different custom commands
- How to optimize tests by creating and restoring user sessions
- How to create automated accessibility tests
- How to use _feature flags_ in automated tests
- How to set up a continuous integration pipeline where tests are run in parallel
- How to configure _test retries_
- How to generate artifacts to help debug failures in continuous integration
